Shopee Affiliate Insight Dashboard 📊

เครื่องมือเว็บแอปพลิเคชันสำหรับนักโปรโมท Shopee Affiliate เพื่อใช้วิเคราะห์รายได้จากไฟล์รายงาน (.xlsx, .csv) อย่างละเอียด โดยเน้นการวิเคราะห์ราย SubID พร้อมการคำนวณค่าคอมมิชชั่นสุทธิหลังหักการคืนสินค้า

คุณสมบัติเด่น (Features)

Multi-file Upload: รองรับการอัปโหลดไฟล์รายงานหลายไฟล์พร้อมกันเพื่อรวมข้อมูล

SubID Deep Dive: วิเคราะห์เส้นทางการคลิก (SubID 1-5) เพื่อดูว่าช่องทางไหนทำเงินได้จริง

Automatic Calculation: - คำนวณยอดขายสุทธิ (หัก 20% รองรับการยกเลิก/คืนของ)

คำนวณค่าคอมมิชชั่นพิเศษ (Extra Bonus) อิงเกณฑ์ 10%

Interactive Visualization: กราฟวงกลม (Doughnut Chart) แสดงสัดส่วนรายได้ของ SubID ทั้งหมด

Multi-format Export: ส่งออกข้อมูลสรุปเป็น Excel, CSV, JSON และ PDF

Quick Copy: ฟีเจอร์คัดลอกตารางเพื่อนำไปวางใน Google Sheets หรือ Excel ได้ทันที

วิธีการใช้งาน (How to use)

ดาวน์โหลดรายงาน "คำสั่งซื้อ" จากระบบ Shopee Affiliate (รองรับทั้งภาษาไทยและอังกฤษ)

เปิดไฟล์ index.html ผ่านเบราว์เซอร์ หรือใช้งานผ่านลิงก์ GitHub Pages

อัปโหลดไฟล์รายงานเข้าสู่ระบบ

ระบบจะสรุปข้อมูลให้เห็นในรูปแบบ Dashboard ทันที

เทคโนโลยีที่ใช้ (Built With)

HTML5 / CSS3 (Tailwind CSS)

JavaScript (Vanilla JS)

SheetJS - สำหรับประมวลผลไฟล์ Excel

Chart.js - สำหรับแสดงกราฟสถิติ

Lucide Icons - ไอคอนประกอบสวยงาม

jsPDF - สำหรับการสร้างไฟล์ PDF

หมายเหตุ: ข้อมูลทั้งหมดถูกประมวลผลบนเบราว์เซอร์ของผู้ใช้เท่านั้น ไม่มีการส่งข้อมูลกลับไปยังเซิร์ฟเวอร์ใดๆ เพื่อความปลอดภัยสูงสุดของข้อมูลรายได้
